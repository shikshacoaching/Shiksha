# Shiksha Student Portal

A professional, lightweight, and installable (PWA) portal for students of Shiksha - A Career Institute of Arts.

## Features
- **PWA Support:** Installable on any mobile device.
- **Real-Time Sync:** Uses Google Apps Script as a secure backend.
- **Fast Loading:** Optimized assets and Service Worker caching.

## Setup
1. Deploy the Google Apps Script as a Web App (Access: Anyone).
2. Copy the Web App URL into `assets/js/script.js` in the `API_URL` variable.
3. Upload this repository to GitHub.
4. Go to **Settings > Pages** and enable GitHub Pages.
